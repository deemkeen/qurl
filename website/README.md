# qURL Website

Modern landing page for qURL - the QR code-based file transfer tool.

## Features

- Clean, modern design with smooth animations
- Fully responsive (mobile, tablet, desktop)
- Animated QR code demo
- Feature showcase
- Installation instructions
- Step-by-step usage guide

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- Vite
- Vanilla JavaScript
- Modern CSS with animations
- SVG graphics

## Customization

Update the GitHub links in `src/main.js` to point to your actual repository.

## Deployment

The built website can be deployed to any static hosting service:
- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

Simply run `npm run build` and deploy the `dist` folder.
